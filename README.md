# 🏠 Moradia Digna

Aplicativo desenvolvido em **Flutter** para a disciplina de **Dispositivos Móveis**, integrando conceitos das disciplinas de **Engenharia de Software** e **Redes de Computadores / Cloud Computing**.

## 📖 Sobre o Projeto

O **Moradia Digna** é um aplicativo criado com o objetivo de demonstrar a utilização de múltiplas telas, navegação entre páginas e organização de informações em uma interface moderna. O projeto aborda conceitos relacionados à arquitetura de sistemas, computação em nuvem e benefícios da integração entre Engenharia de Software e Redes de Computadores.

---

## 🎯 Objetivos

* Desenvolver uma aplicação utilizando Flutter.
* Implementar navegação entre múltiplas telas.
* Aplicar conceitos de Engenharia de Software.
* Demonstrar conceitos de Redes de Computadores e Cloud Computing.
* Criar uma interface moderna utilizando tema escuro.

---

## 📱 Estrutura do Aplicativo

### Tela de Abertura

* Apresentação do aplicativo.
* Botão para acesso ao menu principal.

### Menu Principal

Permite navegar para:

* Arquitetura do Sistema
* Cloud Computing
* Benefícios do Sistema

### Tela 1 – Arquitetura do Sistema

Apresenta conceitos relacionados à Engenharia de Software, enfatizando organização, manutenção e facilidade de uso do sistema.

### Tela 2 – Cloud Computing

Introduz os conceitos de computação em nuvem.

### Tela 2.1 – Comunicação em Rede e Serviços em Nuvem

Aborda:

* Comunicação cliente-servidor;
* Protocolos TCP/IP e HTTP;
* Armazenamento em nuvem;
* Escalabilidade dos serviços.

### Tela 3 – Benefícios do Sistema

Destaca:

* Organização das informações;
* Facilidade de acesso;
* Escalabilidade em nuvem;
* Integração entre Engenharia de Software e Redes.

## 🚀 Como Executar

### 1. Clonar o repositório

```bash
git clone https://github.com/Souza-Ingrid/Moradia-digina.git
```

### 2. Entrar na pasta do projeto

```bash
cd Moradia-digina
```

### 3. Instalar as dependências

```bash
flutter pub get
```

### 4. Executar o projeto

```bash
flutter run
```

Ou executar diretamente no navegador:

```bash
flutter run -d chrome
```
